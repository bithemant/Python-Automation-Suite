# Python Automation & Utility Projects

## Overview
This repository showcases a collection of automation scripts and utility tools developed using Python. The projects focus on applying programming fundamentals to practical use cases including data processing, system automation, web scraping, and file operations.

## Key Areas Covered
- Web Scraping & Data Extraction
- File Handling & Data Conversion (JSON, CSV)
- Encryption & Decryption Utilities
- System-Level Automation Scripts
- CLI-Based Applications
- Productivity Tools

## Objective
To strengthen real-world scripting skills and demonstrate applied Python development through practical implementation.
